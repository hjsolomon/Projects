The program takes in 7 input files: the data for 6 months and the item ids, production costs, and sales prices of items at a coffee shop.
The program outputs 3 files: one with each item's id/total # of sales, one with each item's id, total # of sales, sales cost, and production cost, and 
one with the top 10 earners out of each item

- The code is run with the "make" command in the terminal
- Alternatively, the command "gcc analysis.c functions.c -o analysis.exe" will also work 
if within the designated directory and followed by the command: "./analysis.exe"
